# Viena darba nedēļa datorikas studenta dzīvē

Next.js vienas lapas mājaslapa latviešu valodā par datorikas studenta nedēļu Latvijas Universitātē.

## Palaišana

```bash
npm install
npm run dev
```

Pēc tam atver:

```text
http://localhost:3000
```

## Autora vārds

Autora vārdu var nomainīt failā `app/page.tsx`, tekstā `Autors: Staņislavs Students`.
